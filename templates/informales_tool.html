menu_book Manual: Renumeración de Informales

Esta herramienta automatiza el proceso de asignación de nuevas cédulas catastrales a predios informales basándose en su ubicación geográfica respecto a la capa CTM12.

¿Cómo funciona?
1

Validación: Se corrigen automáticamente problemas de topología (auto-intersecciones) y se eliminan polígonos vacíos.

2

Intersección: Se cruzan los predios informales con los rurales/urbanos del CTM12.

3

Predio Dominante: Si un informal cae en varios CTM, el sistema elige aquel con el que tenga mayor área de contacto.

4

Renumeración: Se genera un consecutivo de 3 dígitos por cada predio CTM y se construye el nuevo código.

warning IMPORTANTE: Archivos ZIP

Debe cargar archivos .ZIP que contengan los componentes del Shapefile (.shp, .shx, .dbf, .prj). El sistema buscará el archivo .shp recursivamente dentro del comprimido.

Salida de Datos
  • Shapefile Resultante: Con campos NumRen y RENUMERADO.
  • Log de Proceso: Archivo JSON con el resumen de cuántos informales se asignaron a cada código matriz.
