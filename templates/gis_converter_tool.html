Especificaciones del Proceso

Esta herramienta extraerá todas las capas vectoriales presentes en la File Geodatabase (.gdb) y las consolidará en un único archivo GeoPackage (.gpkg) estandarizado. Proceso optimizado con Pyogrio.

layers 01_FUNCIONAMIENTO

Convierte estructuras de datos complejas (GDB) en archivos portables de código abierto (GeoPackage). Ideal para compartir datos sin restricciones de software propietario.

warning 02_REQUISITOS

  • -- El archivo subido DEBE ser un .ZIP.
  • -- Dentro del ZIP debe existir la carpeta .gdb original.
  • -- Archivos muy grandes (>500MB) pueden tardar varios minutos.
