description 01_OBJETIVO_NÚCLEO

Este módulo procesa registros de exportación de ancho fijo del Sistema Nacional Catastral (SNC). No requiere saneamiento manual.

layers 02_REGISTROS_SOPORTADOS

R1_SNC (ÍNDICE_BÁSICO)

Datos de propiedad, direcciones y valores de avalúo actuales. Archivo maestro para interrogación del sistema.

R2_SNC (MORFOLOGÍA)

Detalles estructurales: área de terreno, especificaciones de construcción, conteo de pisos y estado de servicios.

RES_LOGS (ALIMENTACIÓN_PROTOCOLO)

Logs de procedimientos internos. Clasificados automáticamente mediante algoritmo de detección de tipo (T1/T2/T3).

tips_and_updates 03_PROTOCOLOS_RECOMENDADOS

  • -- No modifique la fuente FWF en editores de texto enriquecido (potencial corrupción de columnas).
  • -- La codificación Latin-1 asegura la integridad de caracteres para símbolos especiales.
  • -- Descarga inmediata activada tras POST:200.
